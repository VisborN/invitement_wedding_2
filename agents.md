# Контекст для AI-агента (Свадебное Приглашение "Волшебный Лес")

Этот файл содержит краткую сводку по проекту, чтобы любой ИИ-ассистент мог быстро понять архитектуру и продолжить работу без необходимости перечитывать всю историю.

## Описание проекта
Одностраничное интерактивное свадебное приглашение в стилистике "Волшебного леса". Проект реализован без использования тяжелых фреймворков для обеспечения максимальной производительности и полного контроля над анимациями.

## Стек технологий
- **Vanilla HTML5** — семантическая структура, формы.
- **Vanilla CSS3** — кастомные переменные, CSS-анимации, flexbox, parallax эффекты.
- **Vanilla JavaScript (ES6)** — логика навигации, эффекты Canvas, генерация звука.
- **Шрифты** — Google Fonts (Cormorant Garamond, Montserrat).

## Реализованный функционал
1. **Прелоадер и защита от FOUC (Flash of Unstyled Content):**
   - Реализован критический CSS в блоке `<style>` внутри `index.html`.
   - Имеется оверлей `.preloader-overlay`. JS удаляет его с плавным эффектом (`.fade-out`), дожидаясь полной загрузки `document.readyState` (или события `load`).
2. **Процедурные светлячки (Canvas):**
   - Два слоя светлячков: фоновые (мелкие) и передние (крупные, сильно размытые).
   - Светлячки имеют "волнообразную" траекторию, пульсируют (alpha) и реагируют на курсор мыши/касания (мягкое отталкивание).
3. **Процедурный эмбиент леса (Web Audio API):**
   - Отказались от внешних `.wav` файлов из-за ошибок CORB (Cross-Origin Read Blocking).
   - Звук ветра и леса генерируется "на лету" с помощью белого шума, bandpass-фильтров и LFO для имитации порывов ветра. Звук запускается только после первого взаимодействия пользователя со страницей.
4. **Стейт-машина навигации:**
   - Страница разделена на шаги (0 - Welcome, 1 - Details, 2 - Dresscode, 3 - RSVP, 4 - Success).
   - Переключение управляется через изменение атрибута `data-nav-step` у тега `<body>`. CSS отвечает за показ нужных карточек и раздвижение деревьев.
5. **Таймер обратного отсчета:**
   - Отсчет ведется до 12 сентября 2026 года.
6. **RSVP Форма:**
   - Кастомная стилизация чекбоксов и радиокнопок.
   - Имитация отправки данных с сохранением в `localStorage`.
